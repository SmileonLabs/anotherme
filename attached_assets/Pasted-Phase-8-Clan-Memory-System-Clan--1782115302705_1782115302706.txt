Phase 8 — Clan Memory System

목표:

Clan을 단순 랭킹 집단이 아니라, 멤버들의 활동과 성장을 통해 지식·전략·가치관을 축적하는 “집단 기억” 시스템으로 확장합니다.

Clan Memory는 Another Me 서비스의 핵심 차별점입니다.

---

핵심 원칙

* 기존 개인 Persona XP 시스템 수정 금지
* 기존 AI 분석 로직 수정 금지
* 기존 Persona Ranking 로직 수정 금지
* 기존 Clan XP 적립 로직 수정 금지
* 기존 Clan Ranking 로직 수정 금지
* Clan War는 아직 구현하지 않음
* 이번 단계는 Clan Memory 저장/조회/요약까지만 구현
* 실시간 AI 분석 금지
* 버튼 클릭 또는 특정 완료 이벤트에서만 제한적으로 실행

2. Memory 생성 방식

초기 버전에서는 3가지 방식만 구현

1. 수동 기록

* Clan 멤버가 직접 메모 작성
* title, summary, memory_type, tags 입력

2. 던전 완료 기록

* 던전 목표 달성 시 clan memory 후보 생성
* 단, 자동 저장하지 말고 "기억으로 남기기" 버튼으로 저장

3. 토크배틀 승리 기록

* 배틀 승리 후 "가문 전략으로 저장" 버튼으로 저장

주의:
이번 단계에서는 AI가 자동으로 모든 내용을 요약하지 않음

---

3. API 추가

GET /clans/:id/memories

Query:

* type optional
* limit default 30 max 100

POST /clans/:id/memories

Body:
{
"memoryType": "strategy",
"title": "토론에서 먼저 프레임을 잡는 전략",
"summary": "상대의 주장에 바로 반박하기보다, 먼저 기준을 정의하면 설득력이 높아진다.",
"tags": ["토크배틀", "설득", "프레임"]
}

DELETE /clans/:id/memories/:memoryId

권한:

* 생성: clan member 이상
* 삭제: 작성자 또는 elder/owner
