Phase 5 — Clan System

목표:

Another Me 사용자들이 가문에 가입하거나 직접 가문을 만들 수 있는 Clan 시스템을 구현합니다.

Clan은 단순 그룹이 아니라, 비슷한 정체성과 성장 방향을 가진 Another Me들이 모이는 집단입니다.

현재 구현된:

* Persona
* XP 성장
* AI 분석
* Persona Card
* Persona Ranking

기능을 기반으로 Clan 시스템을 추가합니다.

---

핵심 원칙

* 기존 XP 시스템 수정 금지
* 기존 AI 분석 로직 수정 금지
* 기존 Persona Card 로직 수정 금지
* 기존 Ranking 로직 수정 금지
* 이번 단계에서는 Clan War, Clan Memory, Clan Ranking은 구현하지 않음
* Clan 생성/가입/탈퇴/멤버 조회까지만 구현
* 사용자는 한 번에 하나의 대표 Clan에만 소속 가능
* 모바일 우선 UI

---

1. DB 스키마 추가

추가 테이블:

clans

필드:

* id
* name
* description
* emblem_url
* owner_user_id
* level
* exp
* member_count
* clan_values
* clan_summary
* preferred_archetype
* created_at
* updated_at

clan_members

필드:

* id
* clan_id
* user_id
* role

  * owner
  * elder
  * member
* contribution_exp
* joined_at
* updated_at

제약:

* clan name unique
* 한 user_id는 하나의 active clan_members만 가질 수 있음
* owner_user_id는 users와 연결
* clan_members.user_id는 users와 연결
* clan_members.clan_id는 clans와 연결

---

2. Clan 생성 규칙

사용자는 Clan이 없을 때만 생성 가능

생성 시:

* clans row 생성
* 생성자를 clan_members에 owner로 추가
* member_count = 1
* level = 1
* exp = 0

입력값:

* name
* description
* clan_values
* preferred_archetype

기본 문구:

* clan_summary는 description 기반으로 저장
* emblem_url은 optional

---

3. Clan 가입 규칙

초기 버전은 승인제 없이 즉시 가입

조건:

* 이미 Clan에 가입한 사용자는 가입 불가
* 존재하지 않는 Clan 가입 불가

가입 성공 시:

* clan_members row 추가
* clans.member_count +1

---

4. Clan 탈퇴 규칙

일반 member/elder:

* 탈퇴 가능
* member_count -1

owner:

* 멤버가 1명뿐이면 Clan 삭제 가능
* 멤버가 2명 이상이면 탈퇴 불가
* 메시지: "가문장은 다른 멤버에게 권한을 넘긴 후 탈퇴할 수 있습니다."

이번 단계에서는 owner 이전 기능은 구현하지 않아도 됨

---

5. Clan 조회 API

추가 API:

GET /users/me/clan

* 내 Clan 정보
* 내 role
* 멤버 수
* 최근 멤버 일부
* 없으면 null

GET /clans

* Clan 목록 조회
* 검색어 q 지원
* preferred_archetype 필터 optional
* limit 기본 30, 최대 100

GET /clans/:id

* Clan 상세
* 멤버 목록 포함

POST /clans

* Clan 생성

POST /clans/:id/join

* Clan 가입

POST /clans/:id/leave

* Clan 탈퇴

6. Clan UI

화면 추가:

Clan Home 화면

접근 위치:
Another Me 화면 또는 랭킹 화면에서 “가문” 버튼 추가

상태 1: 내 Clan 없음

표시:

* “아직 소속된 가문이 없습니다.”
* “비슷한 정체성을 가진 Another Me들과 함께 성장해보세요.”
* 가문 만들기 버튼
* 가문 찾아보기 버튼

상태 2: 내 Clan 있음

표시:

* Clan 이름
* Clan 설명
* Clan 레벨
* Clan EXP
* 멤버 수
* 내 역할
* 선호 아키타입
* Clan 가치관
* 멤버 일부 목록
* 탈퇴 버튼

---

7. Clan 생성 화면

입력:

* 가문 이름
* 설명
* 대표 가치관
* 선호 아키타입

선호 아키타입 선택:

* 전략가형
* 조율자형
* 탐구자형
* 개척자형
* 현자형
* 엔터테이너형
* 행동가형
* 관찰자형

검증:

* 이름 필수
* 이름 길이 제한
* 설명 길이 제한
* 중복 이름이면 오류 표시

---

8. Clan 목록/검색 화면

표시:

* 가문 이름
* 설명
* 멤버 수
* 선호 아키타입
* 레벨
* 가입 버튼

검색:

* 이름/설명 기준 q 검색
* 아키타입 필터 optional

가입 버튼:

* 이미 Clan이 있으면 비활성화
* 가입 성공 시 내 Clan 화면으로 이동

---

9. 개인정보/보안

Clan 멤버 목록에는 아래만 표시:

* Another Me 이름
* 아바타
* 레벨
* 칭호
* 아키타입
* role
* contribution_exp

표시 금지:

* 이메일
* 전화번호
* 채팅 내용
* AI 분석 상세
* 배틀/던전 원문

---

10. 테스트

확인할 것:

1. Clan 생성 성공
2. Clan 중복 이름 생성 실패
